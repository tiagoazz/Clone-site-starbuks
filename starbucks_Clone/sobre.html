Sobre o Nosso Website

Bem-vindo!

Este website foi criado para oferecer um formulário de cadastro completo e amigável para nossos clientes. Nossa página principal permite que você se registre fornecendo informações pessoais e de contato, com opções específicas para CPF e CNPJ. O formulário foi cuidadosamente projetado para ser fácil de usar e para garantir que todas as informações necessárias sejam coletadas de forma eficiente.

Funcionalidades da Página Principal

  • Escolha entre cadastro com CPF ou CNPJ.
  • Campos específicos que aparecem conforme a seleção de CPF ou CNPJ.
  • Entrada de dados pessoais como nome, e-mail, senha, endereço e telefone.
  • Seleção de cidade e sexo a partir de listas suspensas.
  • Campo para observações adicionais.
  • Botões de enviar, limpar e voltar para facilitar a navegação.

Nossos Objetivos

Nosso objetivo é simplificar o processo de registro e garantir que todas as informações necessárias sejam coletadas de maneira organizada e eficiente. Valorizamos a experiência do usuário e estamos comprometidos em proporcionar uma interface intuitiva e responsiva.
